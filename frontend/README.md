# Frontend

Simple Frontend app that will present a "Say Yes" and "Say No" button.  

It will call back to 2 simple APIs (one `yes` and one `no`) that will return 
amusing Yes and No reasons.  The API results will be displayed in a `<textarea>`
box.  

It is deliberately trivial as the main focus of this larger repo is to work out some thoughts on
mono-repo management w/ Github Actions and CDK.
